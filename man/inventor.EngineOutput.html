inventor.EngineOutput

class EngineOutput(object)

Represents an engine output.

Use this object type to create connections to fields.

Methods defined here:

enable(...)

Enables or disables the connections from this output.

Args:

    True to enable and False to disable connection.

get_container(...)

Returns the engine of this output.

Returns:

    Engine instance that this output is a part of.

get_name(...)

Returns the engine name.

Returns:

    String containing the name under which the output is known in the engine.

get_type(...)

Returns the type of the engine output.

Returns:

    Engine connection type as string.

is_enabled(...)

Returns if connections from this output are enabled.

Returns:

    True if connection is active, otherwise False.
