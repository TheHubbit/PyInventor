PyInventor.QtInventor.QSceneGraphEditor

class QSceneGraphEditor(PySide.QtGui.QApplication)

Application class for a scene graph editor. This application renders scene graph files and displays the scene strcuture in a tree view. It furthermore shows the properties (fields) of the scene objects in a table view. The structure as well as the fields can be modified and changes saved back to file. This class creates an instance of QSceneGraphEditorWindow as main window and links menu entries to it.
